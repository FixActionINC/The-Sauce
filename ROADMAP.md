Product: The Sauce | Type: Client Project | Owner: Tyrone Jones


### Current State Summary

The Sauce is an e-commerce BBQ sauce store built for Tyrone Jones. The platform runs on Next.js 15, Prisma/PostgreSQL, Square payments, Docker/Nginx.

What's built and live:

- Product catalog with categories and filtering
- Shopping cart with persistent state
- Checkout flow with Square payment processing
- Admin panel for product and order management
- Photo gallery showcasing products and events
- Customer testimonials section
- Dockerized deployment with Nginx reverse proxy
